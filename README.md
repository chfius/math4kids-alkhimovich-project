# Математика для детей

### Тема проекта:

Интерактивная обучающая/развивающая программа:

### Идея проекта, описание основной концепции:

В рамках данного проекта будет создано SPA приложение. Идея заключается в интерактивном обучении детей всех возрастов простейшим математическим действиям: сложение, вычитание, умножение, деление. Ребенку будет предлагаться выражение и варианты ответов, выбирая нужный, получаем ответ. Либо он сам составляет выражение и сам же его решает. Чем больше примеров решено, тем более сложные выражения могут быть предложены. За каждый решенный пример в зависимости от сложности дается определенное количество баллов.

### Структура проекта:
#### Главное окно
на главной странице справа находится основное меню с пунктами: Играть, Правила, Учеба, Лучшие результаты
В центре окна находится изображение школьной доски, где будет выводится информация и происходить игровой процесс
#### Страница с описанием правил 
Выводит информацию в поле школьной доски о правилах игры
#### Страница “Учеба” 
содержит описание и примеры основных математических правил
#### Страница с лучшими результатами 
выводит таблицу с именами лучших игроков
#### При нажатии “Играть” 
на школьной доске появляется математическое выражение, где будет предложено:
а) доставить недостающий символ - знак действия, например, “+” или “-”, путем перетаскивания в свободное место предложенного ниже на школьной доске варианта
б) вставить пропущенную цифру в выражении из предложенных ниже
в) самостоятельно составить выражение и решить его
г) в случае правильного решения появляется радостный персонаж, который сообщает о правильном решении
д) в случае ошибки - грустный персонаж с напутствием быть внимательнее. Допускается 3 ошибки. После чего спрашиваем имя игрока и записываем его результат (1 правильный ответ - 1 балл).

### Шаги реализации: 

#### Первая неделя
Составление техзадания.
Утверждение плана работ руководителем проекта.
Определение структуры и рисование эскиза страниц. 
Подготовка контента (текст, изображения и т.п.)
Создание репозитория для выпускного проекта на GitHub, инициализационного коммита с gitignoreв в ветке мастер, создание ветки dev, начало работы над логикой игры.

#### Вторая неделя
Создание игрового поля “школьной доски” на canvas.
Добавление бэкграунда (возможно анимированных персонажей)
Создание анимационных “полей”, которые будут предлагаться как варианты при игре.
Написание правил игры, наполнение страницы “Учеба” информацией.
Добавление звуков при выборе определенных пунктов меню, при выигрыше и проигрыше.

#### Третья неделя
Создание игрового процесса на одном примере.
Обработка варианта правильного ответа, затем ошибочного. В случае неправильного ответа - спросить у пользователя имя и записать результат в таблицу результатов.
Дополнение логики игры возможными вариациями математических выражений.
Проверка поведения игры с использованием анимации.

#### Четвертая неделя
Создание SPA со страницами: Играть, Правила, Учеба, Лучшие результаты.
Добавление сохранения результатов из формы окончания игры в AjaxStringStorage2 и вывод их на страницу Лучших результатов в  виде таблицы
При необходимости добавление в проект jQuery.

#### Пятая неделя
Тестирование и исправление ошибок  и замечаний ( в т.ч. протестировать работу в различных веб браузерах, проверить сайт в соответствии с критериями оценки выпускного проекта).
Подготовка к защите проекта
